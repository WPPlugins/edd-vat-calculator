Calculate automatically the EU VAT on your EDD  checkout page. 

== Description ==

This plugin requires [Easy Digital Downloads](http://wordpress.org/extend/plugins/easy-digital-downloads/) and an active [Quaderno.io](http://quaderno.io) account.

EDD VAT Calculator allows a site owner to calculate automatically the VAT on the EDD checkout page. 

Quaderno.io is an invoice generator and billing management app for Stripe users. Every time you take a payment on Stripe, Quaderno automatically creates and sends a proper invoice to your customers.

== Installation ==

Following are the steps to install the EDD VAT Calculator

1. Unpack the entire contents of this plugin zip file into your wp-content/plugins/ folder locally.
2. Upload to your site.
3. Navigate to wp-admin/plugins.php on your site (your WP Admin plugin page).
4. Activate this plugin.
5. Configure the options from Downloads → Settings → Misc.

OR you can just install it with WordPress by going to Plugins >> Add New >> and type this plugin's name
